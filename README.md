# Config Store

```bash
azc import --account-name '' --account-key '' --file-path ''
azc import -n '' -k '' -f ''
```

```bash
azc export --account-name '' --account-key '' --file-path ''
azc export -n '' -k '' -f ''
```

```bash
azc clear --account-name '' --account-key ''
azc clear -n '' -k ''
```
